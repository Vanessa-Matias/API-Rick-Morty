# Projeto: Webapp Rick and Morty

Este projeto consiste em um Webapp responsivo construído com HTML, CSS e JavaScript, utilizando a API pública do Rick and Morty para exibir e filtrar personagens.

A interface foi customizada com a estética da série (tema escuro e detalhes em verde limão) e inclui funcionalidade interativa com modais para visualização de detalhes dos personagens.
